Accountability: About This Data Set

This dataset contains every weekly Billboard top 100 songs along with their performers from 1958 to 2018, more than 360,000 entries in total. This dataset is from Data.world, a two year old data collaboration platform. And as Billboard record is publicly available, I think this data set is accountable.

Where did this data come from?

https://data.world/kcmillersean/billboard-hot-100-1958-2017

Who has access to it?

It is a public data.
